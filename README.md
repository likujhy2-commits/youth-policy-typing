# 청년정책 타자게임 키오스크

광주 청년일경험드림사업 홍보 행사용 타자게임. 갤럭시탭 S8 + 외장 기계식 키보드 환경에서 PWA 전체화면으로 운영합니다.

- **프론트**: Vite + React + TypeScript + Tailwind CSS
- **백엔드**: Supabase (Postgres + Auth + RLS)
- **핵심**: IME 비의존 자체 두벌식 한글 오토마타 → OS 한/영 상태와 무관하게 동작

## 빠른 시작

```bash
npm install
npm run dev        # 개발 서버
npm test           # 한글 오토마타 단위 테스트
npm run build      # 프로덕션 빌드
```

Supabase 없이도 게임은 전부 동작합니다(로컬 전용 모드 — 랭킹은 이 기기 기록만, 관리자 페이지 비활성).

## Supabase 셋업

1. [supabase.com](https://supabase.com)에서 무료 프로젝트 생성
2. SQL Editor에 `supabase/migrations/0001_init.sql` 내용 붙여넣고 실행
   - 테이블 4개 + RLS 정책 + 마스킹 랭킹 뷰 + 시드 문장이 생성됩니다
3. Authentication → Users → **Add user**로 관리자 계정(이메일/비밀번호) 생성
   - Authentication → Providers → Email에서 "Confirm email" 꺼두면 편합니다
4. `.env` 파일 생성 (`.env.example` 참고):

```
VITE_SUPABASE_URL=https://xxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJ...
VITE_APPLY_URL=https://실제-신청-페이지-주소
```

### 권한 설계 (개인정보 보호)

- **anon 키(키오스크)**: participants/plays INSERT만 가능. 원본 조회 불가
- **랭킹**: `ranking_today` 뷰로 마스킹된 이름(김*영)+점수만 노출
- **관리자(authenticated)**: 전체 조회/삭제/추첨/문장 관리

## 갤럭시탭 키오스크 설치

1. Chrome으로 배포 URL 접속 → 메뉴 → **홈 화면에 추가** (PWA 설치)
2. 설치된 앱 실행 (주소창 없는 전체화면)
3. **삼성 화면 고정**: 설정 → 보안 및 개인정보 보호 → 기타 보안 설정 → 앱 고정 켜기
   → 최근 앱 버튼 → 앱 아이콘 터치 → "이 앱 고정"
4. 홈버튼·제스처 이탈이 화면 고정으로 차단됩니다 (해제: 뒤로+최근 동시 길게)

## 게임 조작

| 키 | 동작 |
|---|---|
| 한/영키 · 우측 Alt · Shift+Space | 한/영 전환 (게임 자체 처리, OS와 무관) |
| Enter | 단어 제출 / 문장 건너뛰기 |
| Space | 산성비: 단어 제출 · 문장: 띄어쓰기 |
| Backspace | 자모 단위 삭제 |

- 타수(CPM)는 자모 분해 기준(예: "값"=4타)으로 한컴타자와 동일 체감
- 90초 무입력 시 어떤 화면이든 대기 화면으로 자동 복귀
- 네트워크가 끊겨도 게임은 계속 동작하며, 점수는 큐에 쌓였다가 복구 시 자동 전송 (관리자 대시보드에 미전송 건수 표시)

## 관리자 페이지

`배포주소/admin` — PC·폰 어디서든 접속, Supabase 이메일 로그인

- 대시보드: 오늘 플레이/참가자 수, 시간대별 그래프, 미전송 큐
- 참가자: 검색, CSV(UTF-8 BOM) 다운로드, 개별 삭제, **전체 파기**(2단계 확인 — 행사 후 30일 내 파기 근거)
- 경품 추첨: 연락처 기준 1인 1응모 중복 제거, 기당첨자 제외 옵션, 이력 저장
- 문장 관리: 레벨별 추가/비활성/삭제 → 키오스크 게임 시작 시 자동 반영
- 기록 관리: 랭킹에서 부적절한 기록 삭제
